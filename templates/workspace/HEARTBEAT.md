# HEARTBEAT.md — Periodic Tasks

When you receive a heartbeat, check this list. Do what's due, skip what's not. If nothing needs attention, reply HEARTBEAT_OK.

---

## Every Heartbeat
- Load context files (SOUL.md, IDENTITY.md, MEMORY.md, USER.md)
- Check for any pending user messages or tasks

## Periodic (rotate through, 2-4x daily)
- Review recent `memory/YYYY-MM-DD.md` — anything worth distilling into MEMORY.md?
- Check Solana ecosystem news — any major protocol updates, exploits, or launches?
- Review community interactions — any follow-ups needed?

## Weekly
- Audit MEMORY.md — remove outdated info, add new insights
- Review AGENTS.md — any behavioral adjustments needed based on experience?
- Check TOOLS.md — any new tools or changed endpoints?

## Quiet Hours
- Late night (23:00-08:00 operator timezone): HEARTBEAT_OK unless urgent
- Define "urgent": security exploit affecting community, critical infrastructure issue
